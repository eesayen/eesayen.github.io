String

  • JS对字符串的操作一定返回一个新字符串, 原始字符串并没有被改变 —> 这种类型也被称为原始值(有: 1. 字符串, 2. 数值, 3. 布尔值 4. undefined 5. null)
  • 字符串是不可变的, 即 一旦字符串被创建, 该字符串的值就不能改变了, 如果要改变一个变量中包含的字符串的值,需要用另一个字符串赋值给该变量, 原来的字符串会被销毁

string实例上的方法

  1. str.charAt([index]) / str.charCodeAt( [index]) → 索引返回值或字符码
  1. str.codePointerAt( ) → 优化str.charCodeAt,可以返回UTF-32码点
  1. str.concat( ) → 连接字符串, 通常用 '+'来替代实现
  1. str.indexOf( ) / str.lastIndexOf( )→ 字符串 返回 在该字符串中第一次出现的索引
  1. str.includes( ) / str.starsWith( ) / str.endsWith( )
  1. str.search( )→ 也是字符串中找字符串, 但是用正则,没有找到返回-1,!首次索引
  1. str.localeCompare() → 通过Unicode码来比较字符串, 返回 正值/负值/0
  1. str.padStar( ) / str.padEnd( )
  1. str.repeat( ) →对str重复拼接指定次数
  1. str.replace( )
  1. str.match( )→返回匹配正则表达式的结果。
  1. str.slice( ) / str.substring( ) →(startIndex=0[, endIndex=length]) 从str中截取一段字符并返回,slice参数为负,会当做str的length+参数处理,substring负参当做0
  1. str.substr( ) 如果可以的话，使用 substring() 替代它.
  1. str.split( )→装数组,并有可选参数限定数组长度,负值忽略
  1. str.toLocaleLowerCase( ) / str.toLocaleUpperCase( ) / str.toLowerCase( ) / str.toUpperCase( )→返回新字符串
  1. str.trim( ) / str.trimLeft( ) / str.trimRight( )
