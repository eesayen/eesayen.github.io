vue - vue-loader - webpack 模板

  • 这个样板是针对大型，专业的项目，需要熟悉Webpack和vue-loader

  • vue-loader是一个 webpack的加载器，是你能够以 单个文件组件（SFCs single-file components） 的形式编写vue
  • vue-loader 有许多强大的特性
    1. 可以为Vue组件中的每一部分使用不同的加载器， 比如：<style> 用Sass， <template>用Pug
    1. 可以在vue文件中，可以自定义块并提供提供自定义的加载器链
    1. 将<style> <template>中引用的静态资源作为当前组件的依赖，将这些资源交给webpack加载器处理
    1. 模拟仅限当前组件范围的CSS作用域
    1. 在开发过程中，保存时热重载

webpack 和 Vue-loader 的结合，为前段工作提供了一个现代化，弹性且极其强大的 Vue应用编写

安装 Vue-loader

  • 用vue-cli脚手架 + webpack 模板 初始化一个vue项目，会预先配置好vue-loader和其他的常用开发需要
  • 手动安装
    • vue-loader 和 vue-template-compiler 需要同时安装
    npm install -D vue-loader vue-template-compiler

    它们要分别安装，因为vue-template-compiler的发布与vue版本的发布是一起的，并且vue-template-compiler和vue包必须同步，
