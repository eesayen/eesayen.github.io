简介

NPM → node package manager

  • npm 的发展是跟 Node.js 的发展相辅相成的。

Node.js → 链接js这门语言,让它跑到操作系统上

Node.js 是由一个在德国工作的美国程序员 Ryan Dahl 写的。他写了 Node.js，但是 Node.js 缺少一个包管理器，于是他和 npm 的作者一拍即合、抱团取暖，最终 Node.js 内置了 npm。

  • 本来只能跑到浏览器上,有了Node之后,可以作为一个桥梁,链接了操作系统和JS这门语言 ,直接调用操作系统上的一些接口,做到JS以前不可以做到的一些事情: 比如文件操作,

NPM只负责将包下载到本机文件,不负责在项目中引入

安装 NodeJs,自带npm
